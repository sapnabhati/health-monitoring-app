# AuraHealth AI

AuraHealth AI is an AI-powered health monitoring platform that tracks both mental and physical wellness.

## 🚀 Features
- Real-time health tracking (heart rate, sleep, activity, stress)
- AI-based mental health analysis
- Personalized recovery plans
- Interactive chatbot support
- Secure health reports

## 🛠 Tech Stack
- React + TypeScript + Vite
- Python (Backend)
- Machine Learning

## 📌 Description
This project focuses on providing a smart, predictive, and user-friendly healthcare solution using AI.
